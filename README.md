# Mphakathi - Voice Detection & Emotion Monitoring

This repository contains the source code for Mphakathi, an intelligent voice detection and emotion monitoring app.

## Run Locally

**Prerequisites:** Node.js

1.  Install dependencies:
    `npm install`
2.  Set the `GEMINI_API_KEY` in a `.env` file to your Gemini API key.
3.  Run the app:
    `npm run dev`
